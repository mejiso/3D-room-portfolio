# Interactive 3D Personal Portfolio

An interactive 3D personal portfolio designed in Blender and built with React, TypeScript, Three.js, React Three Fiber, HTML, and CSS. This project transforms my real-life room into a web experience where users can explore different parts of my background through clickable 3D objects, camera transitions, music, and interactive UI panels.

## Live Demo

[View Portfolio](https://mejiso.dev)

## Overview

This portfolio was created to make my personal website feel more immersive, creative, and memorable. Instead of using a traditional scrolling layout, I modeled a 3D version of my real-life room in Blender and turned it into an interactive environment. Each object in the room acts as a navigation point that highlights a different part of my background, including my experience, projects, education, music, clubs, and personal interests.

The goal of this project was to combine front-end development, 3D design, and storytelling into one cohesive portfolio experience.

## Features

- Custom 3D room modeled in Blender
- Interactive hotspots for navigating portfolio sections
- Smooth camera transitions between room objects
- Experience, education, projects, clubs, and fun facts sections
- Dropdown-based content organization for cleaner readability
- Music player with original tracks
- Spinning vinyl player animation
- Responsive mobile layout
- Playful hover animations on 3D text
- Styled pop-up information panels
- Deployed through Vercel

## What I Learned

Through this project, I learned how to combine 3D design with front-end development to create an interactive web experience. I gained more experience working with React Three Fiber, camera positioning, 3D asset placement, responsive design, state management, and UI layering.

I also learned how to design a portfolio as both a technical project and a personal storytelling platform. This project helped me think more intentionally about user experience, visual hierarchy, interaction design, and how to present my background in a creative but professional way.

## Challenges

Some of the biggest challenges included:

- Positioning 3D hotspots accurately across local and deployed environments
- Maintaining consistent camera views across different screen sizes
- Optimizing the mobile layout without affecting desktop styling
- Balancing 3D visuals with readable UI panels
- Integrating music controls without overcrowding the interface
- Keeping the room visually consistent with the overall pastel/cartoon aesthetic
